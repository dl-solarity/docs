# EC256

## Overview

#### License: MIT

```solidity
library EC256
```

Cryptography module

Elliptic curve arithmetic over a 256-bit prime field (Weierstrass curve y^2 = x^3 + ax + b (mod p)).
## Structs info

### Curve
